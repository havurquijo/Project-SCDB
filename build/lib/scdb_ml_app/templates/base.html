Project SCDB

We analise the U.S. Supreme Court Database using some machine learning algorithms (MLA). Particularly we found a 96% accuracy rate in the predictions of the Decision Direction variable when used a Decision Tree Classifier MLA on a 9 variables set which don't depend on the Decision Direction.
